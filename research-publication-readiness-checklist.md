# Research Publication Readiness Checklist

This checklist ensures public research assets remain structured for future journal submission without making false publication claims.

## Metadata

- [ ] SEO title and meta description
- [ ] Canonical URL
- [ ] Open Graph tags
- [ ] Twitter card tags
- [ ] JSON-LD `Article` schema
- [ ] `author`, `datePublished`, `dateModified`

## Structure

- [ ] Executive summary / abstract
- [ ] Scope and target audience
- [ ] Methodology or evidence basis
- [ ] Main sections with heading hierarchy
- [ ] Tables/frameworks readable on mobile
- [ ] Limitations and assumptions
- [ ] Compliance caution and policy-change caution

## Policy and Ethics

- [ ] No acceptance/indexing claims unless formally true
- [ ] No legal/compliance advice framing
- [ ] No PHI or confidential source data

## Citation Discipline

- [ ] External references listed
- [ ] DOI/CMS references validated
- [ ] Suggested citation block present if needed

## Public UI ORCID Rule

- [ ] ORCID metadata can remain in repo files
- [ ] ORCID labels/badges/sections are absent from public pages
